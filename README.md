# Increased Node Loot

## Description

Increases the node loot from gathering (mining & herbing) by a multiplier specified in the config files.

Does not increase gem loot from mining nodes.

![](/screenshots/my_module.png?raw=true)

## Requirements

increased-node-loot requires:

- AzerothCore v3.0.0+


## Installation

```
1) `git clone` the module under the `modules` directory of your AzerothCore source or copy paste it manually.
2) Re-run cmake and launch your server.
```

## Configuration file location

If you want to change the loot multiplier, go to your server configuration directory (where your `worldserver` or `worldserver.exe` is), copy `increased_node_loot.conf.dist` to `increased_node_loot.conf` and edit that new file.
